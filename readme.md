## Prerequisites

- CMake (version 3.0.0)
- C++ compiler (supporting C++11 or later)
- OpenCV (version 4.9.0)
## Getting Started

### Clone the Repository

```bash
git clone https://github.com/Flame25/ROS_OpenCV_THT_Aksantara.git
cd ROS_OpenCV_THT_Aksantara
```
### Buld  the Project

```bash
cd build
cmake ..
make
./THT_Soal8 <input video>
``
